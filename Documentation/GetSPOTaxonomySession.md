#Get-SPOTaxonomySession
Returns a taxonomy session
##Syntax
```powershell
Get-SPOTaxonomySession
        [-Web <WebPipeBind>]
```


##Parameters
Parameter|Type|Required|Description
---------|----|--------|-----------
|Web|WebPipeBind|False|The web to apply the command to. Omit this parameter to use the current web.|
