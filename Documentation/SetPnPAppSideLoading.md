# Set-PnPAppSideLoading

## SYNOPSIS
Enables the App SideLoading Feature on a site

## SYNTAX 

### On
```powershell
Set-PnPAppSideLoading -On [<SwitchParameter>]
```

### Off
```powershell
Set-PnPAppSideLoading -Off [<SwitchParameter>]
```

## PARAMETERS

### -Off


```yaml
Type: SwitchParameter
Parameter Sets: Off

Required: True
Position: Named
Accept pipeline input: False
```

### -On


```yaml
Type: SwitchParameter
Parameter Sets: On

Required: True
Position: Named
Accept pipeline input: False
```
