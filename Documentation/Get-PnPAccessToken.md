# Get-PnPAccessToken

## SYNOPSIS
Returns the current OAuth Access token

## DESCRIPTION
Gets the OAuth 2.0 Access Token to consume the Microsoft Graph API

## EXAMPLES

### ------------------EXAMPLE 1------------------
```powershell
PS:> Get-PnPAccessToken
```

Gets the OAuth 2.0 Access Token to consume the Microsoft Graph API
